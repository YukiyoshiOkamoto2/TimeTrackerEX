# timetracker/ - TimeTracker固有設定

TimeTracker機能の固有設定。

## 設定項目

- **baseUrl**: TimeTracker APIベースURL
- **projectId**: プロジェクトID
- **userName**: ユーザー名
- **scheduleInputInfo**: スケジュール入力設定
- **eventInputInfo**: イベント入力設定
- **timeOffEvent**: 休暇イベント設定
- **isHistoryAutoInput**: 履歴自動入力有効化
